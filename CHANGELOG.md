# Changelog

All notable changes to **Chocolate Themes** will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/)
and adheres to the principles of
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---

## [0.1.0] - 2026-01-12

### Initial Release

### Added
- **Chocolate · Dairy Milk Dark**
  - Premium purple and gold color palette
  - Optimized for long coding sessions
  - Full UI coverage including title bar, activity bar, sidebar, tabs, and terminal
- **Chocolate · KitKat Dark**
  - Bold red and cream palette inspired by KitKat
  - High contrast for better readability
  - Enhanced bracket matching and selection colors
- **Chocolate · Snickers Dark**
  - Warm chocolate, caramel, and peanut-inspired palette
  - Comfortable dark tones with subtle blue accents
  - Balanced contrast for extended use

### Features
- Semantic highlighting enabled for modern language servers
- TextMate token support for backward compatibility
- Consistent UI styling across all themes
- Carefully tuned colors for accessibility and reduced eye strain
